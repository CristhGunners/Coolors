Coolors
=======

Beautiful Color Palette.

**Stack**

 - HTML5 (Jade)
 - CSS3 (Stylus)
 - Javascript (JQuery)
 - Atom Editor
 - Gulp.js

**60 Palettes**

[Go Coolors](http://cristhgunners.github.io/Coolors/)

**Get Started**

- `git clone git@github.com:CristhGunners/Coolors.git coolors`
- `cd coolors`
- `npm install`

**Running in development**

- `gulp serve`
- Open [http://localhost:3000/](http://localhost:3000/)
